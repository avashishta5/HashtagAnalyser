# HashtagAnalyser

A simple script to analyse Hashtags from twitter
## Installation
```
$ git clone https://github.com/avashishta5/HashtagAnalyser.git
$ cd HashtagAnalyser
$ pip3 install requirements.txt
```
## Usage
```
$ python3 analyse.py "Hashtag-Name"
```
